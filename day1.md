# 머신러닝과 딥러닝 기초 이해
머신러닝 : 규칙을 일일이 프로그래밍하지 않아도 자동으로 데이터에서 규칙을 학습하는 알고리즘을 연구하는 분야
- 파이썬 API 기반 사이킷런 라이브러리 활용, 머신러닝 프로그램 제작 가능

딥러닝 : 머신러닝 알고리즘 중 인공 신경망을 기반으로 한 방법들을 통칭하여 일컫는 말
- ex. 알파고
- 구글의 Tensorflow와 페이스북의 PyTorch는 대표적인 오픈소스 딥러닝 라이브러리
